Introduction To JavaScript - History

JavaScript is a programming language which was designed and built to enable programmers to implement complex features on web pages. It was designed and built by a language engineer called Brendan Eich.
Brendan Eich built JavaScript in 10 days and at the time he was hired by a company known as Netscape. He built it in 1995. The prototype that he came up with was crude and raw but it worked and had the fundamentals of a programming language.
Through the contributions of another programmer called Guy Steele, the language was standardized and the crudeness was removed. When Brendan Eich built it, he called it MOCHA. Then he changed the name to LIVESCRIPT and finally he renamed it JavaScript.
Note that JavaScript is different from Java. They are two completely different languages owned by different organisations.

JavaScript and ECMA

JavaScript is built around the ECMA standard. It was initially designed to give dynamism and interactivity to the client side of static HTML pages. It was also initially designed for Netscape 2. In 1997 however, JavaScript adopted the standard of European Computer Manufacturers Association (ECMA).
A language library or standard was built around ECMA features and that standard was called ES (ECMA - Standard). Once this was done, Netscape handed over JavaScript to ECMA to manage but retained the right to develop the version that a firefox browser will be needing.
The first ES version was developed in 1997 and it known as ES1. In 1998 ES2 was developed and in 2000, ES3 was developed. Currently, we are in ES10/12, but the most stable version is ES6, though some companies still use ES5.
Today, several libraries have been built around JavaScript and its ES standards. These includes

  • ReactJS (used for frontend development and for making API calls)
  • NextJS (a framework built around ReactJS and used to integrate the frontend with the backend)
  • ReactNative (used for mobile applications development)

All major browsers and search engines today run a JavaScript engine which itself is built around ES standards. These major search engines include:


  • Internet explorer / microsoft edge
  • Mozilla firefox
  • Google Chrome
  • Google
  • Yahoo
  • Adobe
  • Safari
  • Opera

JavaScript is an Event - Driven Languge

An Event - Driven Language is one whose instruction sense are governed by events

  • the action of a keyboard
  • the action of a mouse

These two devices (a mouse and a keyboard) and the events they create constitute the triggers that drive functions in JavaScript. A mouse event can be

  • a click
  • double click
  • a hover
  • a pointer etc.

A keyboard event can be any action taken with the keyboard. For example

  • enter button
  • shift key
  • backspace etc.

Application of JavaScript

Though JavaScript was initially created to give dynamism to the frontend static pages of the web. It can today work with a browser and without a browser. When it works with a browser, it integrates with HTML and CSS and does the following

  • It creates real time content updates
  • It makes API calls
  • It links frontend to backend
  • It gives dynamism and interactivity to web pages
  • It performs routing
  • It handles 2D and 3D animations and graphics

Along with HTML and CSS therefore, JavaScript constitutes the third leg in the basket of technologies that power the web globally. Outside the web, JavaScript can be used to create applications in the areas of machine learning and artificial intelligence. It provides a server side runtime environment for executing its codes outside the browser and it does this using Nodejs.

Outside of HTML and CSS, JavaScript can be used to develop several other application types. Examples include

  • Server - side applications using NodeJS
  • Desktop applications using a framework called Electron
  • Mobile applications development using ReactNative framework and NativeScript.
  • Game development using any of the following frameworks - Web-GL, Phaser and/or ThreeJS

The language In Which JavaScript is Written

JavaScript was originally written in C language, but its modern implementation is in C++ language.JavaScript is a full programming language and so it complements HTML which is a markup language and CSS which is used for styling.
HTML and CSS are not programming languages. As a markup language for example, most of HTML's syntax and contents sit in between tags and HTML uses hyperlinks to connect one page to another. HTML is browser based and it is used to create static content and structure on a web page.
On the otherhand, CSS is a stylesheet that uses a cascading method to execute instructions. It is also browser based and has to reference HTML tags and elements to effectively style its content. Like HTML, CSS builds statics pages.
JavaScript is however different from HTML and CSS. It is a full programming language. It is built around the following features:

  • Data Type
  • Data Structure
  • Algorithms
  • Variables
  • Functions
  • Objects
  • Methods
  • Arrays
  • Conditional Statements
  • Loops

Let us look at some of these terms and their meanings;


  1. Data Type - Data type refers to the category that every piece of data received is structured into. Data can be numbers for example, it can be whole numbers (integers) or it can be numbers with decimals or floating points (floats).
    That can also be a STRING (a collection of characters - lowercase alphabets, uppercase alphabets and symbols like ?, !, etc).
    Data can also be BOOLEAN (either / or, True / False, Yes / No).
  2. Data Structure - Once data have been categorized into a data type, it has to be organized into a data structure to make it programmable. There are different types of data structures;
    • Arrays
    • list
    • Linked List
    • Graph
    • Stack
    • Queue
  3. Algorithms - These refers to the sequence of steps that a piece of code follows as it executes. There are different standard algorithms;
    • Sorting
    • Search
    • Divide and Conquer etc.
